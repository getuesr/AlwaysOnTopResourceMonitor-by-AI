# Py 浮动窗口资源监视器
这个是DeepSeek写的基于py浮动窗口性能监视器(浮动窗口在活跃窗口上方) 原因是我找到很少的类似资源监视器 让AI写了一份 您可以下载 然后修改它 增强它 它应该被每个人免费的修改使用！
📦 依赖库清单
库名	最低版本	用途
PyQt5	5.15.0	创建浮动窗口、托盘图标、界面渲染
pywin32	305	调用 Windows API（获取窗口句柄、进程信息、注册表操作）

指令安装 pip install PyQt5 pywin32

exe是打包好的 直接运行就可以了 目前没有图标 托盘是小圆点 可以退出 或者开机自启动

This is a floating window performance monitor (floating window above the active window) written by DeepSeek based on Python. The reason is that I found very few similar resource monitors, so I had AI write one. You can download it and then modify it, enhance it. It should be free for everyone to modify and use!
📦 Dependency List
Library Name | Minimum Version | Purpose
PyQt5 | 5.15.0 | Create floating window, tray icon, interface rendering
pywin32 | 305 | Call Windows API (get window handle, process info, registry operations)

Command to install: pip install PyQt5 pywin32

The exe is packaged and can be run directly. Currently, there is no icon. The tray shows a small dot. You can exit or set it to start automatically on boot.
